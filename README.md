# Doctor365

A modern telemedicine platform that connects patients with healthcare providers for remote consultations, appointment scheduling, and symptom analysis.

## Overview

Doctor365 is a full-stack healthcare application built with a microservices architecture. It provides a seamless experience for patients to consult with doctors, manage appointments, process payments, and receive personalized health recommendations. The platform combines telemedicine capabilities with intelligent symptom checking to deliver accessible healthcare services.

## Features

- **Patient Management** - Secure patient profiles with medical history and contact information
- **Doctor Services** - Doctor profile management and availability scheduling
- **Appointment Booking** - Real-time appointment scheduling and management
- **Telemedicine** - Live video consultations using Agora SDK
- **Symptom Analyzer** - AI-powered symptom checking and preliminary health analysis
- **Payment Processing** - Integrated payment gateway for service transactions
- **Notifications** - Real-time alerts for appointments and service updates
- **User Authentication** - Secure JWT-based authentication system
- **API Gateway** - Centralized request routing and orchestration
